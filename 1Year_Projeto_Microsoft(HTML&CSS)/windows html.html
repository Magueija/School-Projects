O Windows está constantemente a melhorar

O Windows 10 Fall Creators Update expande a sua criatividade e produtividade com novas realidades envolventes, histórias contadas em 3D, maior ligação aos seus contactos e experiências entre dispositivos. As atualizações são implementadas automaticamente quando o seu PC estiver preparado; alternativamente, pode transferir o Assistente de Atualização.
